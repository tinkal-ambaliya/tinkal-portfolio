## 🚀 How to View Your Portfolio

### Option 1: Open Locally
1. Double-click on `index.html`
2. It will open in your default web browser
3. Navigate using the menu or scroll down

### Option 2: Use Live Server (Recommended)
1. If you have VS Code, install the "Live Server" extension
2. Right-click on `index.html` and select "Open with Live Server"
3. This allows you to see changes in real-time

## 📸 Adding Project Images

You'll notice placeholder images for projects. Here's how to add your own:

### For Capital Valley Marketing Project:
1. Take screenshots of your WordPress prototype
2. Save them as `project-wordpress.jpg` in this folder
3. The website will automatically display them

### For Database Project:
1. Take screenshots of your ER diagrams or database queries
2. Save them as `project-database.jpg` in this folder
3. They'll appear on the portfolio automatically

### For Excel Project:
1. Screenshot your Excel dashboards
2. Save as `project-excel.jpg`

**Tip:** Keep image files under 1MB for faster loading. You can use online tools to compress images.

## ✏️ How to Customize

### Changing Your Information:
1. Open `index.html` in a text editor (VS Code, Notepad++, etc.)
2. Find the section you want to edit
3. Update the text between the HTML tags
4. Save and refresh your browser

### Changing Colors:
1. Open `styles.css`
2. Find the `:root` section at the top
3. Change the color values (they're in hex codes like #FFB5D8)
4. Save and refresh

### Adding More Projects:
1. Copy an existing project card in `index.html`
2. Update the title, description, and tags
3. Create a new detail page (copy `project-database.html` as a template)

## 📧 Making the Contact Form Work

Currently, the form shows an alert when submitted. To make it actually send emails:

1. **Use Formspree** (Easy, Free):
   - Go to formspree.io
   - Create an account
   - Get your form endpoint
   - Update the form action in `index.html`

2. **Use EmailJS** (Free):
   - Go to emailjs.com
   - Follow their setup guide
   - Add their JavaScript to send emails

3. **Backend Solution** (Advanced):
   - Use PHP, Node.js, or Python to handle form submissions
   - Requires web hosting with backend support

## 🆘 Need Help?

Common issues:

**Images not showing?**
- Make sure image filenames match exactly (case-sensitive)
- Check that images are in the same folder as index.html

**Colors look wrong?**
- Clear your browser cache
- Make sure you edited styles.css, not project-details.css

**Links not working?**
- Make sure all files are in the same folder
- Check for typos in filenames

**Site looks broken?**
- Make sure all 3 CSS files are in the same folder
- Check that styles.css and project-details.css are linked in the HTML
